Para ejecutar el sitio web es necesario:
	1. En laragon, en el directorio www se deben meter los
	proyectos de bloomrecipes y proyectobloom.

	2. En laragon, en la base de datos mariadb se debe importar
	la base de datos que está en el archivo basededatosbloom.sql
	(totalmente necesario pues tiene los datos de todas las recetas, categorias y mas).
	Esto se hace desde Archivo -> Ejecutar archivo sql 
	
	3. Despues de que la base de datos está llena se puede abrir el
	proyecto bloomrecipes que es la página web
